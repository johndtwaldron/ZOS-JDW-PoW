# ZOS-JDW-PoW

Personal z/OS / COBOL / mainframe power-of-work (PoW) repo.

## Structure

- `Cobol/`     – COBOL course notes, cheat sheets, examples
- `JCL/`       – JCL snippets, job templates, utilities
- `Assembler/` – HLASM notes and small examples
- `REXX/`      – REXX scripts and notes
- `SYSPROG/`   – System programming, JES, SDSF, SMP/E, IPL, etc.
- `SMF/`       – SMF records, performance, accounting notes
- `DUMPs/`     – Abend analysis, IPCS, debugging notes

## Goal

A high-level but practical crib sheet and knowledge base
to operate as a solid z/OS COBOL/legacy programmer, sysprog,
developer, morderniser and infrastructure specialist.
